2. Supported Platforms

KSCD explicitely supports the following platforms:

  • Linux
  • FreeBSD
  • NetBSD
  • BSD386
  • Sun
  • Solaris (including cdda support)
  • HPUX
  • SGI Irix (including cdda support)
  • Sony NEWS
  • OSF/1
  • Ultrix
and should compile on many others with few modifications.
